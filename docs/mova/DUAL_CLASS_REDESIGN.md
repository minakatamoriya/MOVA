# MOVA 双职业重设计方案

本文档保留为旧双职业方向草案，不等同于当前版本终稿，也不等同于当前已实装内容。

当前最新方向：双职业天赋已被从目标成长结构中整体删除。本文档只保留作历史草案参考，不再作为后续实现依据。

当前已调整的结论：
- 本文档中的 30 组“主职业 + 副职业”拆分方案不再作为当前版本主方向。
- 双职业组合技能后续只按 15 种唯一职业组合收束设计。
- 主职业天赋、副职业天赋维持现状，优先保证当前成长结构稳定。
- 深度专精移至主职业树末端，作为主职业终局内容。

相关现状请先看 [SKILL_LIST.md](SKILL_LIST.md)。

## 原始目标（已归档）

- 旧方案曾尝试按“主职业 + 副职业”分别设计，因此扩张为 30 组，而不是 15 组。
- 双职业的第一职责是补主职业短板。
- 双职业的第二职责是让副职业系统真正接入主循环。
- 双职业尽量不新增主动按钮，优先通过规则改变、触发联动、区域联动来建立差异。

## 统一模板

每组双职业固定设计 3 个 3 级天赋：

- 补板天赋：优先解决主职业最明显的玩法短板。
- 转轴天赋：让副职业系统稳定介入主循环。
- 成型天赋：让整组 build 最后形成一句清晰的玩法描述。

## 主职业短板总表

### 法主

- 站桩要求高。
- 被贴脸时容错低。
- 持续压场依赖站位。

### 猎主

- 正面承压差。
- 群体压制依赖后期成型。
- 缺稳定控场。

### 战主

- 远端处理弱。
- 对分散敌群覆盖差。
- 非低血阶段节奏不够饱满。

### 骑主

- 主动追击能力偏弱。
- 输出上限偏慢热。
- 敌人不打你时部分价值递减。

### 术主

- 起爆慢。
- 对高速/分散敌人容易漏伤。
- 前中期需要时间滚雪球。

### 德主

- 即时爆发反馈偏弱。
- 近身高压处理一般。
- 输出强但节奏容易发散。

## 法主双职业

### 法主 + 猎副

- 冰痕追猎：被寒霜减速的敌人更容易被假人箭矢追猎，并逐级强化猎印与碎冰处决。
- 游猎施法：移动施法损失降低，逐级减少移动时冰弹输出折损。
- 霜羽处决：冻结、猎印、碎冰形成稳定收尾链。

### 法主 + 战副

- 冰甲血性：低血时获得更稳的站桩和反打窗口。
- 寒刃断筋：近身目标更容易被留在法师的控制区内。
- 极寒战意：血怒与寒霜爆炸联动，残血阶段输出显著抬升。

### 法主 + 术副

- 寒疫扩散：毒圈中的敌人更容易被寒霜控制与碎冰补伤。
- 冰骨前线：骷髅为法师争取站桩空间，并延长控场收益。
- 霜灾领域：碎冰与毒域互相接力，形成区域压杀。

### 法主 + 骑副

- 圣棱护幕：法师站定时获得更强保护与减伤。
- 寒霜审判：冻结目标会被附加审判，放大控制收益。
- 圣域咏唱：法师从脆皮站桩转为据点炮台玩法。

### 法主 + 德副

- 星熊护法：熊为法师提供前排代偿并稳定贴身环境。
- 冰落共鸣：星落补寒霜层，碎冰缩短星落循环。
- 极地星域：冻结区与坠落区合并成强控制场。

## 猎主双职业

### 猎主 + 法副

- 附魔箭列：箭列附带小额激光追击，强化清后排能力。
- 法阵步猎：法阵内射击更稳、更快，并强化移动输出。
- 棱镜齐射：炮台与猎印联动，形成稳定点杀链。

### 猎主 + 战副

- 猎战步伐：提高近身环境容错与断筋反贴脸能力。
- 血怒猎杀：低血时箭列进入明显处决节奏。
- 穿心处刑：让猎人更擅长斩杀低血目标。

### 猎主 + 术副

- 毒羽猎场：毒圈补足猎人的区域持续伤害。
- 白骨掩护：骷髅帮助猎人顶住前线并稳输出线。
- 腐羽收割：持续伤害负责压线，箭列负责收头。

### 猎主 + 骑副

- 圣羽庇护：补猎人的减伤、格挡与持续站场能力。
- 裁决箭幕：箭幕附带神圣伤害与击退压制。
- 猎誓反制：把受击后的防守节奏转为强化箭幕反打。

### 猎主 + 德副

- 熊卫箭阵：熊灵为猎人挡线，保证输出环境。
- 鹰狩同频：战鹰优先追猎印目标，强化 Boss 战节奏。
- 林野齐鸣：树精与自然恢复抬高猎人持续输出容错。

## 战主双职业

### 战主 + 法副

- 霜刃外放：剑气获得减速和更强远端追击价值。
- 奥术旋壁：战士前排绞杀，炮台负责补远端和后排。
- 棱镜断阵：近战与法阵前后压制，补分散敌群处理。

### 战主 + 猎副

- 断筋猎场：假人和断筋把敌人留在旋刃击杀区内。
- 破风猎杀：剑气更擅长追击与低血收头。
- 战猎围剿：聚怪、锁定、绞杀形成完整围剿玩法。

### 战主 + 术副

- 毒刃绞场：战士近身区与毒圈叠成持续掉血区。
- 白骨压锋：召唤物负责补边线与漏怪处理。
- 腐灭冲阵：战士冲入毒域中心后能越打越滚。

### 战主 + 骑副

- 圣钢战躯：提升近身开团容错与稳态减伤。
- 审判旋锋：旋刃命中触发审判，扩张近身禁区。
- 十字处决：把击退、审判、低血收割缝成连续终结链。

### 战主 + 德副

- 铁木前驱：熊灵与战士共担前线压力。
- 鹰刃追魂：战鹰补战士远端处理与追残血。
- 先祖狂潮：树精为战士补平滑续航和血怒维持。

## 骑主双职业

### 骑主 + 法副

- 圣棱重锤：锤击与炮台互相缩短输出循环。
- 冰审禁区：审判区获得更稳定的控制与减速。
- 法盾裁决：格挡、防守与法阵火力合流，补主动伤害。

### 骑主 + 猎副

- 猎誓制裁：猎印让圣骑更好打点杀与追击。
- 盲雨审判：箭雨与神圣冲击合并为压场技能组。
- 反击号令：把被动反制转成主动追杀。

### 骑主 + 战副

- 血怒圣裁：骑主低血期进入明显爆发窗口。
- 战吼压阵：不靠挨打也能主动启动增伤节奏。
- 不屈惩戒：补足圣骑的低血终结能力。

### 骑主 + 术副

- 圣腐疆域：毒圈与审判区叠成双重地面压制。
- 骸骨护行：召唤物帮助圣骑推进并提高锤击命中率。
- 审判蚀刻：神圣回击与腐蚀磨血形成稳定处刑链。

### 骑主 + 德副

- 圣林庇佑：熊灵与树精让圣骑更稳推进。
- 星锤共振：锤击命中驱动星落与区域伤害。
- 林地裁决：圣骑从纯反制变成稳步碾压型前线。

## 术主双职业

### 术主 + 法副

- 寒疫棱镜：炮台帮助术士补起爆和清后排。
- 冰骨术式：毒圈中的敌人更容易被控住。
- 奥灾叠界：毒域与法阵重叠形成据点玩法。

### 术主 + 猎副

- 诱饵瘟域：假人把敌人留在毒圈里。
- 猎腐处决：猎印帮术士补残血终结。
- 腐羽围猎：毒域压线、箭矢收网，形成围猎玩法。

### 术主 + 战副

- 血腐战躯：补术士贴脸时的反打空间。
- 断筋疫场：近身敌人更难逃离毒域。
- 腐灭冲锋：地狱火与低血反打构成前压毒阵。

### 术主 + 骑副

- 白骨圣障：白骨护甲与圣印形成双保险。
- 腐蚀审判：审判目标更适合被毒伤磨杀。
- 罪火禁区：地狱火、审判、毒圈构成稳守中心区。

### 术主 + 德副

- 腐林共生：熊顶前线，树精补续航，战鹰补收尾。
- 凋零滋养：宠物与腐蚀状态形成稳定互喂。
- 枯荣王庭：毒域、伙伴、地狱火合成阵地流终局。

## 德主双职业

### 德主 + 法副

- 星棱导引：法阵帮助星落更稳定收束与补伤。
- 寒星余震：补德主控制和区内减速。
- 天穹法环：法阵与坠落区融合成轰炸据点。

### 德主 + 猎副

- 猎星陷阵：假人把敌人固定在星落区内。
- 鹰眼天坠：猎印帮助德主补 Boss 点杀。
- 箭雨星幕：箭雨与陨落形成高频覆盖链。

### 德主 + 战副

- 铁星护体：补贴脸环境的生存与反压能力。
- 血怒星火：残血时星火与陨石更有处决感。
- 陨坠断筋：让德主拥有更明确的近身控制层。

### 德主 + 术副

- 腐星雨：星落区补持续毒伤。
- 召魂丰壤：召唤物为德主提高命中率和前场稳定性。
- 天灾腐林：星落与毒域叠成持续灾变玩法。

### 德主 + 骑副

- 圣林庇护：补德主高压环境生存与续战。
- 审判陨落：星落命中附加审判，强化压区。
- 光种天幕：圣域与星落合并为稳守轰炸流。

## 落地规则

- 每组双职业只允许 1 个新的核心状态词，避免系统爆炸。
- 优先复用现有对象：炮台、假人、熊、鹰、树精、骷髅、地狱火、圣印、审判、毒圈、星落、旋刃、箭环、激光。
- 优先做“规则改变”，少做纯数值包。
- 旧方案里即使扩张到 30 组，也不建议全部做成新增主动技能；自动触发、区域联动、目标规则改变会更适合当前项目。

## 建议优先级

第一梯队优先落地：

- 法主 + 术副
- 猎主 + 法副
- 战主 + 骑副
- 术主 + 德副
- 德主 + 术副
- 骑主 + 猎副

第二梯队：

- 法主 + 德副
- 猎主 + 德副
- 战主 + 法副
- 骑主 + 术副
- 术主 + 法副
- 德主 + 法副

其余组合可按通用模板批量推进。
